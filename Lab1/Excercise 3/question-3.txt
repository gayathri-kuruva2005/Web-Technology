Exercise 3: HTML5 Form Elements 
Create an Advanced Registration Form using HTML5 
Create a user registration form that uses HTML5-specific input types. Include the following fields: 
•	Full Name (text) 
•	Email (email) 
•	Date of Birth (date) 
•	Phone Number (tel) 
•	Password (password) 
•	Choose Profile Picture (file) 
•	Checkbox to accept Terms & Conditions 
Use required validation wherever appropriate.
